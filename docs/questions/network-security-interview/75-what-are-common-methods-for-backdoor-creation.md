# Q: What are common methods for backdoor creation?

**A:** Common methods for creating backdoors include:

1.  **Backdoor Program:** Installing a malicious program to maintain access.
2.  **Backdoor Account:** Creating a hidden user account.
3.  **Backdoor Service:** Installing a service that can be remotely accessed.
4.  **Backdoor Port:** Opening a hidden network port.
5.  **Backdoor File:** Creating a file that can be executed to gain access.
6.  **Backdoor Network:** Setting up a covert network channel.
7.  **Backdoor Script:** Writing a script that can be executed to provide access.
8.  **Backdoor Replacement:** Replacing a legitimate program with a trojanised version.
