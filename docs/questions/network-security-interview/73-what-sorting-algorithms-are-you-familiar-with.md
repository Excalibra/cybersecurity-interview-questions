# Q: What sorting algorithms are you familiar with?

**A:** Sorting algorithms can be broadly categorised into internal and external sorts.

**Internal Sorting Algorithms (In-Memory):**
- Bubble Sort
- Insertion Sort
- Selection Sort
- Shell Sort
- Merge Sort
- Quick Sort
- Heap Sort

**External Sorting Algorithms (For Large Data):**
- External Sorting
- Record Aggregation Sort
- Bucket Sort
- File Merge Sort
