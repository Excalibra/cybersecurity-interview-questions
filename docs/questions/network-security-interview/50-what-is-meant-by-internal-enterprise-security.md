# Q: What is meant by internal enterprise security?

**A:** Internal enterprise security refers to all the measures an organisation takes to secure its internal information and systems. This includes:

1.  **Security Policies:** Defining roles, responsibilities, and rules for security.
2.  **Security Measures:** Implementing technical and administrative controls.
3.  **Security Organisation:** Establishing a security team with clear responsibilities.
4.  **Security Training:** Ensuring staff are educated on security best practices.
5.  **Vulnerability Assessments:** Regularly scanning for internal risks.
6.  **Incident Handling:** Having a process to investigate and resolve internal breaches.
7.  **Performance Monitoring:** Continuously monitoring security effectiveness.
8.  **Policy Updates:** Keeping security policies up to date.
9.  **Security Awareness Programs:** Promoting a security-conscious culture.
10. **Security Logging:** Maintaining detailed logs of security-related events for auditing and analysis.
