# 21. What are the dangerous functions for PHP code execution and PHP command execution?

## PHP Code Execution
- `call_user_func()`
- `call_user_func_array()`
- `create_function()`
- `array_map()`

## PHP Command Execution
- `system()`
- `shell_exec()`
- `passthru()`
- `exec()`
- `popen()`
- `proc_open()`
- `putenv()`
