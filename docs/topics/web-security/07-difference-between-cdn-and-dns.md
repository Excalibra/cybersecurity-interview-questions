# 7. What is the difference between CDN and DNS? What are the approaches for bypassing CDN?

## CDN (Content Delivery Network)
Its primary function is to enable users to access web resources from a geographically proximate node, improving response speed and reducing network congestion.

## DNS (Domain Name System)
Its primary function is to translate domain names into IP addresses.

## CDN Bypass Approaches
- Subdomain enumeration
- Internal email headers
- Dark search engine queries
- Pinging from foreign locations
- Certificate and DNS records analysis
- App traffic interception
- Misconfiguration exposure
- Scanning the entire IP range
- DoS attacks
