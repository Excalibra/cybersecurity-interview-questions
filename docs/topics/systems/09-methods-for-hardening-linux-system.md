# 9. What are the methods for hardening a Linux system?

1. Prohibit direct root login via SSH configuration
2. Enforce minimum password length of 8 characters
3. Lock account for 5 minutes after 3 failed logins
4. Restrict `su` to root and `wheel` group
5. Disable ICMP echo response
6. Set idle login timeout to 10 minutes
7. Terminate illegitimate login sessions
