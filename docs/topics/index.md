# Topics

Welcome to the **Topics** section. This is a curated collection of foundational cybersecurity topics, organised by category.

## Categories

- [Computer Networks](/docs/topics/computer-networks/) – OSI model, TCP/IP fundamentals.
- [Incident Response](/docs/topics/incident-response/) – Threat intelligence, log analysis, malware detection, honeypots.
- [Internal Network Penetration](/docs/topics/internal-network-penetration/) – Lateral movement, domain controller attacks, golden/silver tickets, tunneling.
- [Systems](/docs/topics/systems/) – Windows/Linux hardening, privilege escalation, persistence detection.
- [Tools](/docs/topics/tools/) – Nmap, sqlmap, webshells, proxies, tunneling tools.
- [Traceability and Traffic Analysis](/docs/topics/traceability-and-traffic-analysis/) – Wireshark, traffic patterns, attack attribution.
- [Web Security](/docs/topics/web-security/) – SQLi, XSS, CSRF, SSRF, deserialization, file uploads, logic flaws.
